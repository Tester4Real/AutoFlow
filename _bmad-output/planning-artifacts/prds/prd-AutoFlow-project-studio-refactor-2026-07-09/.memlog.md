updated: 2026-07-09T10:09

- (decision) Create PRD for project-based AutoFlow refactor using project-context and brainstorming memlog as primary inputs.
- (decision) Top-level product concept is Project Studio: full extension page for projects, assets, JSON mapping, image review, and video queue preparation.
- (decision) Side panel remains operations cockpit for selected project, connection/account status, current queue, run/stop/retry, import entry, and opening Project Studio.
- (decision) JSON-listed references are required; missing or unresolved references block affected prompts until user manually uploads/resolves assets.
- (decision) Videos are not generated automatically after images; selected image variants create draft/ready video jobs that user manually queues/runs.
- (decision) Use stable internal IDs for projects, assets, prompts, variants, and jobs; display names and filenames are mutable labels, not identity.
- (event) PRD finalized
